Side Scroller

  • by: Justin Bauer
  • date: Spring 2021
  • technologies: ARM, C

In my Computer Organization course (ECE243) at UofT, my partner and I developed a side scroller game similar to Mario for the ARM processor using C. We created all the animations from scratch and took input directly from the keyboard buffer. We swapped memories between each frame to create smoother animations. The game is a simple 2D platformer level where the user jumps over obstacles and collects coins before reaching the end.

Below is a demo of our game in action. Apologies for the lag in the video, the game runs smoother without video capture running.
