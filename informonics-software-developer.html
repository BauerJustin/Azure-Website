Informonics - Software Developer

  • June 2020 - August 2020
  • technologies: Angular, Python, Amazon Web Services

During the summer of 2020, I completed a 12-week remote internship developing Informonics' data extraction tool and a couple of side projects. I designed and implemented functionality to improve the accuracy of the Optical Character Recognition (OCR) data extraction tool, using AWS Textract and AWS Lambda with Python. I streamlined the document query page by developing custom pagination, using Angular and AWS DynamoDB. This enabled users to filter and sort documents without loading all documents at once. I designed a document preview with ability to zoom and change pages, accessing images from AWS S3 with Angular. In order to organize documents into groups, I developed a document tagging system by processing stored tags from AWS S3 in an AWS Lambda function written in Python and implementing the organization in Angular. I remodelled OCR algorithm to lower scan costs by replacing algorithm with open-source OCR software. Finally, I created a user login page for an ASP.NET application using C# and HTML linked to a SQL database.
