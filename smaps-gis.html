sMaps Geographic Information System

  • by: Justin Bauer
  • date: Spring 2021
  • technologies: C++

In my Software Communication and Design course, ECE297, at UofT, I developed a Geographic Information System using C++ in a team. Our map called sMaps, is mapping software designed for students coming fresh to University campuses. Some of our key features include a weather widget, in depth point of interest information, and the enhanced path finder.

Below is a demo of our point of interest feature. When a point of interest is clicked on the map, it is highlighted in purple and information is provided in the terminal window.

Below is a demo of the enhanced path finder. This feature includes a path finding algorithm using the A* method which can be activated by clicking on the map or typing directions in the autocomplete search bars. Directions are provided to the user in a pop up window.
